## Another Mouse mover

Key functionality achieved using [robotjs](https://github.com/octalmage/robotjs).

Due to ESM, node 22 is required. 

### Build

Install [bun](https://bun.sh/) first.

```shell
bun install
bun run build
```

Result binary will be at `dist/amm`

### Run

```shell
dist/amm
```

Show debug logs

```shell
LOG_LEVEL=debug dist/amm
```
